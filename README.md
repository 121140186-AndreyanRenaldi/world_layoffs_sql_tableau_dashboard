# Project Overview

This project analyzes global layoff data using SQL for data cleaning and exploratory data analysis (EDA), followed by an interactive dashboard built in Tableau. The objective is to uncover trends in layoffs across companies, industries, countries, and time, providing meaningful business insights through data visualization.

# Business Problem

This project aims to answer the following business questions:

1. Which companies experienced the highest number of layoffs?
2. Which industries were most affected by layoffs?
3. Which countries recorded the highest layoffs?
4. How did layoffs change over time?
5. Which company stages were most impacted?
6. Is there any relationship between funding raised and the number of layoffs?

# Dataset

1. Dataset : World Layoffs
2. Period  : 2020 - 2023
3. Record  : 2,361 (after data cleaning)
4. Format  : CSV

## Key Insights
1. January recorded the highest number of layoffs, with **92,037** employees affected.
2. The **United States** experienced the highest number of layoffs, totaling **256,559** employees.
3. The **Consumer** industry was the most affected, with **44,782** layoffs.
4. **Amazon** recorded the highest number of layoffs among all companies, with **18,150** employees affected.
